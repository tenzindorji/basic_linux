# basic_linux
Find Kernal version:
```
uname -a
```
Find current IP address
Old way:
```
ifconfig
```
New way:
```
ip
```
How to check free disk space:
```
df -ah
```
How do you manage services on a system?
Newer system
```
systemd
```
Older system:
```
service <service_name> stop/start/status/restart
```
